# Contributing

Local Web Fix is a live, student-run small-business service. I keep changes accurate, accessible, and consistent with the published scope.

1. Create a branch from `main`.
2. Run `npm install` and `npm test`.
3. Keep dependencies at zero unless a documented requirement justifies one.
4. Check mobile layout, keyboard navigation, focus visibility, and both colour themes.
5. Do not add testimonials, client names, results, guarantees, contact details, or new services without documented permission and evidence.
6. Do not expose payment links in public navigation or request passwords through forms.

Use UK English, short paragraphs, plain language, and a mobile-first layout. Avoid extra JavaScript unless it solves a real interaction requirement.
